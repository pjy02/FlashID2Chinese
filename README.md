## 使用步骤:

1. 运行build.bat编译或从Release中下载FlashID2Chinese.exe。

2. 将FlashID2Chinese.exe移动至FlashID安装目录。

3. 以管理员身份运行FlashID2Chinese.exe。

4. 完成。
